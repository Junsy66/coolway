公司简介
北京金日恒升科技有限公司是国家认证的高新技术企业，全球领先的地铁隧道视频媒体方案解决商和网络运营商， 目前运营总部和研发基地设在北京金日科技园，下设全资机构：北京金日酷媒文化传播有限公司，上海酷媒广告有限公司+固安。 北京金日恒升科技有限公司是中国首家自主运营的地铁隧道视频媒体公司。2003年，公司以前瞻性的战略眼光投资研发地铁隧道视频媒体系统。2005年，第一代地铁隧道视频媒体系统研发成功，并在北京地铁试运营，同期展开市场化运作。 作为一家拥有自主知识产权的高科技公司，公司始终致力于将最先进的信息技术与隧道视频媒体系统有效结合，协助地铁公司提升运营质量，并为地铁乘客提供更为丰富的信息服务。 目前北京、上海、深圳、成都等城市每日有超过 1800 万人次通过金日隧道媒体获取信息，极大丰富了地铁乘客的出行生活。

惠金所互金平台
惠金所互金平台
关键词:  投资理财   个人消费   融资通道

      惠金所互金平台是致力为个人、企业、金融机构提供金融资产交易信息服务，打造而成的综合性互联网金融服务平台。该平台为广大投资者提供多样化的投资理财选择，为个人消费和中小微企业提供专业、高效、便捷的融资通道，为非标金融资产提供透明便捷的转让平台，为发展普惠金融、改善民生、促进消费、推动中小微企业发展、提供金融资产的流动性、支持创新型国家建设发挥积极的作用。

      从2015年12月到2016年1月，我们团队仅仅用两个月的时间，实现了惠金所平台的整体上线。为提高运营效率，平台同时提供一系列的基础服务，例如：系统监控和预警、业务监控和预警、流量监控和管理、业务分析报表等。平台自上线至2017年一季度末，平台累计交易量达到423.36亿元，2017年第一季度交易量达到134.67亿元，平台上线以来，实现了零交易失误记录。随着市场的不断拓展，惠金所的产品线不断丰富，平台的交易规模屡创新高。
